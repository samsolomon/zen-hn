# Zen HN

A calmer way to read Hacker News. Less noise, more focus.

Zen HN is a Chrome extension that restyles Hacker News and provides additional controls.

![Zen HN Screenshot](assets/screenshot.png)

## Features

- **Random** - Stumbleupon for HN. Discover random posts from the past.
- **Keyboard navigation** - Keyboard focused design allows navigating HN entirely via keyboard.
- **Comment collapse** - Quickly expand/collapse comment threads.
- **Search** - Algolia search palette with `/` key.
- **External site escape** - Quickly return to HN from external links.
- **Appearance controls** - View HN how you like with hundreds of appearance combinations.


## Appearance

- **Color modes** - Dark and light modes with the option to sync with your system.
- **Themes** - 20+ color themes that support light, dark and contrast color modes.
- **Typography** - 10+ font families in 5 size options.
- **Content width** - 5 width presets.
- **Contrast Mode** - Normal, high contrast or sync with system.


## Keyboard Shortcuts

- **Comment and lists** - Arrow keys to move, Enter to open, Space to collapse.
- **Actions** - Quick keys for upvote, favorite, copy link and more.
- **Go-to shortcuts** - Chord shortcuts for navigating between pages.
- **Help** - Press `?` to see all shortcuts.


## Accessibility

- Screen reader support with live announcements
- High contrast mode
- Full keyboard navigation
- Respects reduced motion preferences

## Install

- Download or and unzip this repo.
- Open `chrome://extensions` or go to manage extensions in Chrome.
- Enable Developer mode.
- Click "Load unpacked" and select the repo folder.
