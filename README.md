# zen-hn

Zen HN is a Chrome extension that restyles Hacker News for a calm, more readable experience.

![Zen HN Screenshot](assets/screenshot.png)

## Features

- **Random** - Stumbleupon for HN. Selects a random post from the past.
- **Keyboard Navigation** - Vim-inspired navigation for posts and comments using arrow keys and shortcuts.
- **Comment Collapse** - Expand/collapse comment threads with Space or click.
- **Search** - Algolia search palette with `/` key.
- **External Site Escape** - Press Esc to return to HN from external links.
- **HN Settings** - Toggle showdead, noprocrast directly from user settings.


## Appearance

- **Color Modes** - Light, Dark, or sync with your system.
- **Themes** - 22+ color themes.
- **Typography** - 11 font families and 5 size options.
- **Content Width** - 5 width presets from Narrow to Full.
- **Contrast Mode** - Normal, High, or sync with system.


## Keyboard Shortcuts

- **Navigation** - Arrow keys to move, Enter to open, Space to collapse.
- **Actions** - Quick keys for upvote (u), favorite (f), copy link (l).
- **Go-to shortcuts** - Chord shortcuts starting with 'g' (g+h for home, g+n for new, etc.).
- **Help** - Press `?` to see all shortcuts.


## Accessibility

- Screen reader support with live announcements
- High contrast mode
- Full keyboard navigation
- Respects reduced motion preference


## Install

⚠️ Warning: This a work in progress.

- Download or and unzip this repo.
- Open `chrome://extensions` or go to manage extensions in Chrome.
- Enable Developer mode.
- Click "Load unpacked" and select the repo folder.
